# 📊 Customer Feedback Intelligence System

A Machine Learning and Natural Language Processing project that automatically analyzes customer reviews to extract business insights.

The system performs:

- Sentiment Analysis
- Complaint Categorization
- Topic Extraction
- Customer Satisfaction Analysis
- Feedback Intelligence Reporting

This project simulates a real-world customer feedback analytics platform used by modern companies.
